# Gurleen's Personal Chatbot

This is a personalized chatbot created using **FAISS** (Facebook AI Similarity Search) and the **Sentence Transformers** library. It allows users to chat with a bot that can answer questions related to **Gurleen's** projects, skills, education, and more.

## Features:
- A chatbot that provides answers to questions about Gurleen, based on personal data.
- Built using **Sentence Transformer** for converting user queries into vector embeddings.
- Uses **FAISS** to perform similarity search and find the best match for user queries.
- Frontend built with **Streamlit** for an interactive user interface.

## Technologies Used:
- **Python**: For backend processing and chatbot logic.
- **Streamlit**: For the frontend web interface.
- **Sentence Transformers**: For converting user queries into embeddings.
- **FAISS**: For storing embeddings and performing similarity search.
- **JSON**: For storing personal data (such as projects, skills, education, etc.) used by the chatbot.
